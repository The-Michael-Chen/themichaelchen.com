# Project Images

Add project images to this folder if you want local assets instead of remote links.

The website supports multiple images per project. Update [data/site-data.js](/Users/michaelchen/Development/Personal%20Website/data/site-data.js) and add more objects to a project's `images` array:

```js
images: [
  { src: "assets/example-1.jpg", alt: "Project image one" },
  { src: "assets/example-2.jpg", alt: "Project image two" },
]
```
